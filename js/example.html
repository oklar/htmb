HTMB Examples

Basic Elements

div({ class: "container" }, h1("Hello World"), p("Welcome to HTMB") )

Nested Elements

div({ class: "card" }, h2({ class: "title" }, "Card Title"), div({ class: "content" }, p("Some text here"), p("More text here") ), button({ disabled: true }, "Disabled Button") )

List Helper

const items = ["Apple", "Banana", "Cherry"]; div( h3("Shopping List"), list(items, item => div({ class: "item" }, span({ style: "color: #666" }, "• "), item ) ) )

Style Helper

div( style(".custom { color: blue; }"), p({ class: "custom" }, "Blue text") )
